# vue-h5-template

## 针对移动端H5的vue框架

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production or uat
```
npm run build:uat
```

```
npm run build:pro
```

### Lints and fixes files
```
npm run lint
```

### 备注
```
ui框架:vant
vue:2.6+
支持sass，scss
添加移动端相关测试，转换插件
```
